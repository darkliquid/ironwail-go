# Interface

## Main surface

Observed validation surface:
- fixture directories under `cmd/qgo/testdata`
- round-trip helper `loadVM`
- unit tests for compiler helpers and end-to-end compilation

## Contracts

- fixture directories are valid compiler inputs and should remain small, readable examples of supported language features
- round-trip tests must be able to load generated bytes into `internal/qc.VM`
- helper-level tests pin allocator, string-table, and opcode-selection behavior so backend regressions fail near the source
